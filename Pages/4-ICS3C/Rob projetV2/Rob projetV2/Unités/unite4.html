Unité 4: Chronomètre et mouvements en C#
Dans cette unité, tu vas commencer à programmer avec le langage C# sur l'interface Visual Studio. Tu pourras télécharger la version de Visual Studio community de Microsoft gratuite. Vous apprendrez comment faire interagir une personne avec votre application. Vous verrez comment: - Utiliser des variables pour manipuler de l'information, entre autres, des variables aléatoires - Utiliser des fenêtres surgissantes de messages - Utiliser des fenêtres pour entrer donner de l'information à votre application - Faire un jeu simple interactif
